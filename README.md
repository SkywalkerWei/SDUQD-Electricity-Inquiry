# SDUQD-Electricity-Inquiry
山东大学青岛校区 宿舍电费查询工具

不再需要进入山大v卡通即可查询宿舍当前电费余量

## 前置条件

- pip install requests
- 我使用的是python 3.11，不保证向下兼容
- 目前应该可以查S1 S2 S5 S6 S7 S8 S9 S10 S11 B10 T1 T3

## 使用方法

1. 运行 main.py 即可
2. 同级目录下打开控制台，输入python main.py 运行
